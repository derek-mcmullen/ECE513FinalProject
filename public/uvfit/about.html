About UV Fit

This is the beginnings of a fully functioning web portal that will support the access and modification of UVFit devices. We are creating this web page for ECE 513 at the University of Arizona

UVFit is a IoT connected device built on the photon microcontroller. It interfaces with a UV sensor and GPS unit to collect Location, Speed, and UV readings for a user tracking their activity levels.

It provides a user with the ability to initiate an activity from the device and remotely log all relevant data. The server is then able to calculate activity summaries while providing an easy to use interface for tracking a users route!
